## How to add a language / Как добавить язык:

**Ru:**
1. Откройте папку `Creator` (она есть всегда).
2. Внутри создайте папку `langs` (её нет по умолчанию, так как `ru` и `en` уже вшиты в `.exe`).
3. Скачайте готовый языковой файл (например, `en.json`) из папки `Creator/langs` исходного кода программы.
4. Переименуйте файл по коду языка, например `es.json` (испанский).
5. Откройте файл и замените текст в строках. Формат: `"ключ": "перевод"`.

**En:**
1. Open the `Creator` folder (it always exists).
2. Inside it, create a folder named `langs` (not present by default because `ru` and `en` are built into the `.exe`).
3. Download a language file (e.g., `en.json`) from the `Creator/langs` folder in the source code / repository.
4. Rename the file by language code, e.g., `es.json` (Spanish).
5. Open the file and replace the text in the lines. Format: `"key": "translation"`.
